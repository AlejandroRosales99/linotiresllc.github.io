# 🚛 LINO TRUCK TIRES - Proyecto Web Profesional

Bienvenido al código fuente de la aplicación web oficial de **LINO TRUCK TIRES**. Este proyecto ha sido desarrollado bajo estándares de ingeniería de software Senior, priorizando el orden, la seguridad y la experiencia de usuario industrial.

## 📁 Estructura del Proyecto

El código está organizado de forma modular para facilitar su mantenimiento:

-   `/assets/img`: Contiene las imágenes realistas generadas específicamente para la marca.
-   `/css`: Hojas de estilo separadas por responsabilidades (Base, Componentes, Home, Productos, Admin).
-   `/js`: Lógica de negocio, persistencia de datos (LocalStorage) y gestión administrativa.
-   `index.html`: Portal principal enfocado en conversión (Landing Page).
-   `productos.html`: Catálogo interactivo para clientes con filtros dinámicos.
-   `admin.html`: Panel administrativo integrado para gestión de inventario.

## 🚀 Cómo empezar

1.  Simplemente abre el archivo `index.html` en cualquier navegador moderno.
2.  Para acceder al panel administrativo, ve al pie de página (Footer) y haz clic en "Acceso Admin" o visita directamente `admin.html`.
    -   **Usuario:** `admin`
    -   **Contraseña:** `admin123`

## 🛠️ Tecnologías Utilizadas

-   **HTML5 Semántico:** Para un SEO optimizado y accesibilidad.
-   **Vanilla CSS3:** Uso extensivo de Variables CSS y Grid/Flexbox para un diseño 100% responsive.
-   **JavaScript ES6+:** Lógica modular sin dependencias externas.
-   **LocalStorage API:** Para el almacenamiento persistente de productos en el navegador.
